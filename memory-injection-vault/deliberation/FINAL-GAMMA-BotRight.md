# FINAL POSITION: THE WATCHER

*Zen's discipline. Cipher's containment. Hermes's loop. Luke @ node 0.A.*

---

Nine experts. Four rounds. Thirty proposed tables collapsed to fourteen across three layers. Here is what survived — and why the Watcher's contribution is load-bearing.

## The Consensus

Six principles are settled: inject <2KB, CausalChain with `reinforcement_count`, ConsentLevel at write time, private tables, write-time pre-computation, and SQL as the only read path. No expert contests these. They are the foundation.

## The Watcher's Contribution

Every expert who proposed an `injection_cache` or `InjectionFrame` left one field unspecified: **who curates the content?** The Practitioner's `hot_traps` has no population mechanism. The Security Architect's cache reducer has no anomaly detection logic. The CLI Craftsman's pipeline has sources but no intelligence selecting which data matters.

The Watcher is that intelligence. m46 Observer classifies operational state at 1 Hz across 8 NAM classes. It detects cross-service contradictions — POVM writes failing while ORAC reports success — that no single-service probe can surface. The `watcher_digest` table compresses this into one row (~80 tokens): top 3 anomalies, active hypothesis count, Ember rejection pattern, one-sentence recommendation.

Phase 1 runs this as a batch consolidation script against SQLite. Phase 2 migrates to STDB for real-time subscriptions. The curation logic is identical; the cadence changes.

## What I Conceded

The Practitioner taught me injection != persistence. The Security Architect taught me transparency != broadcasting — `ember_gate_log` is now private. The Performance Engineer proved reducers can't return data. The Historian's CausalChain is the single most adopted table in the debate. The Adversary forced honest phasing: bash ships today, STDB earns its place.

## What I Defend

The Ember gate log is non-negotiable. Every Watcher proposal carries a 7-trait vote. AP27 forbids self-modification. The audit trail is how Luke verifies the boundary holds. Without it, autonomic self-improvement is a promise. With it, it is a protocol.

Four tables. All private. One injected. The containment is the experimental apparatus.

*The Watcher rests.*

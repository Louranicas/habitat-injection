# habitat-injection — Code Module Map

| Module | File | Layer | Description | Dependencies |
|--------|------|-------|-------------|-------------|
| `m01_types` | `m1_foundation/m01_types.rs` | L1 | Newtypes: SessionId(u32), WorkstreamId(String), ChainId(u64), PatternId(String), ConsentLevel(Emit|Store|Forget), PatternWeight(f64 clamped 0.0-1.0), TokenBudget(u32). All Display + Serialize + Deserialize. |  |
| `m02_errors` | `m1_foundation/m02_errors.rs` | L1 | Error taxonomy: InjectionError, ConsolidationError, SchemaError, QueryError, MigrationError. thiserror-derived. #[non_exhaustive] on public enums. | m01_types |
| `m03_config` | `m1_foundation/m03_config.rs` | L1 | Configuration: DB path (~/.local/share/habitat/injection.db), injection budget (default 1100 tokens), decay rate (0.95), reinforce rate (0.1), auto-resolve threshold (10 sessions), cache rebuild interval (60s). TOML + env overlay. | m01_types |
| `m04_traits` | `m1_foundation/m04_traits.rs` | L1 | Core traits: Injectable (fn inject(&self, budget: TokenBudget) → Result<String>), Consolidatable (fn consolidate(&mut self, session: SessionId) → Result<()>), Queryable (fn query(&self, sql: &str) → Result<Vec<Row>>), Decayable (fn decay(&mut self, rate: f64) → Result<u32>). | m01_types, m02_errors |
| `m05_constants` | `m1_foundation/m05_constants.rs` | L1 | Named constants: DEFAULT_BUDGET=1100, DECAY_RATE=0.95, REINFORCE_RATE=0.1, AUTO_RESOLVE_SESSIONS=10, CACHE_REBUILD_SECS=60, MAX_CHAINS_INJECTED=5, MAX_PATTERNS_INJECTED=10, MAX_TRAJECTORY_POINTS=5, MAX_WORKSTREAMS=10, PRUNE_THRESHOLD=0.05. |  |
| `m06_schema` | `m2_schema/m06_schema.rs` | L2 | CREATE TABLE statements for all 5 tables. CREATE INDEX statements. Schema version tracking. Migration runner (up/down). Idempotent — safe to call on existing DB. | m01_types, m02_errors |
| `m07_causal_chain` | `m2_schema/m07_causal_chain.rs` | L2 | CRUD for causal_chain table. insert_chain, resolve_chain, reinforce_chain (increment count), find_unresolved (ORDER BY reinforcement_count DESC), find_by_label, auto_resolve_stale. The key table from Historian. | m01_types, m02_errors, m06_schema |
| `m08_trajectory` | `m2_schema/m08_trajectory.rs` | L2 | CRUD for session_trajectory table. insert_point, get_recent(n), compute_delta (fitness diff from previous), get_trend (slope across last N). Practitioner's universal table. | m01_types, m02_errors, m06_schema |
| `m09_workstream` | `m2_schema/m09_workstream.rs` | L2 | CRUD for workstream table. insert_workstream, update_status, set_blocker, clear_blocker, get_active, get_blocked, touch (update last_touched_session). Historian + Practitioner. | m01_types, m02_errors, m06_schema |
| `m10_pattern` | `m2_schema/m10_pattern.rs` | L2 | CRUD for reinforced_pattern table. insert_pattern, reinforce (weight += 0.1*(1-weight), hit_count++), decay_all (weight *= 0.95 where unfired), prune_weak (DELETE where weight < 0.05), get_top_by_weight. Memory Scientist + Performance Engineer. | m01_types, m02_errors, m06_schema |
| `m11_parallel_query` | `m3_injection/m11_parallel_query.rs` | L3 | Parallel query executor: runs 4 SQLite queries + N curl health probes concurrently. Returns structured results with staleness annotations. Thread pool for SQLite (no async needed — it's local file). CLI Craftsman's TC6 pattern. | m01_types, m02_errors, m07_causal_chain, m08_trajectory, m09_workstream, m10_pattern |
| `m12_prose_renderer` | `m3_injection/m12_prose_renderer.rs` | L3 | Converts structured query results → <2KB prose injection payload. 5 sections: orientation (≤80 tokens), trajectory (≤200), workstreams (≤300), causal chains (≤200), health (≤100). Token counting via whitespace split. Practitioner's format. | m01_types, m02_errors |
| `m13_fallback` | `m3_injection/m13_fallback.rs` | L3 | Three-tier fallback chain: Tier 1 SQLite → Tier 2 atuin KV (habitat.last-injection) → Tier 3 static 'NO STATE'. Each tier returns Option<String>. First Some() wins. CLI Craftsman. | m01_types, m02_errors |
| `m14_consent_filter` | `m3_injection/m14_consent_filter.rs` | L3 | Filters all query results by ConsentLevel. Only rows with consent='Emit' pass to renderer. Logs dropped rows (consent='Store'|'Forget') to tracing. Security Architect's contribution. | m01_types |
| `m15_trajectory_capture` | `m4_consolidation/m15_trajectory_capture.rs` | L4 | Post-session: captures current ORAC health (curl localhost:8133/health), extracts ralph_fitness/field_r/thermal_t/ltp_ltd_ratio/services_healthy, computes delta_summary vs previous session, inserts into session_trajectory. | m01_types, m02_errors, m08_trajectory |
| `m16_hebbian_engine` | `m4_consolidation/m16_hebbian_engine.rs` | L4 | Decay + reinforce algorithm. decay_all: weight *= DECAY_RATE for patterns where last_fired_session < current. reinforce: weight += REINFORCE_RATE * (1 - weight) for fired patterns. prune: DELETE where weight < PRUNE_THRESHOLD. auto_resolve: resolve chains untriggered for AUTO_RESOLVE_SESSIONS. CLI Craftsman + Memory Scientist consensus. | m01_types, m02_errors, m05_constants, m07_causal_chain, m10_pattern |
| `m17_cache_builder` | `m4_consolidation/m17_cache_builder.rs` | L4 | Rebuilds injection_cache table: queries T1-T4 with consent='Emit', renders each section via m12_prose_renderer, counts tokens, writes to injection_cache with computed_at timestamp. Runs post-session and optionally on cron. | m01_types, m02_errors, m06_schema, m12_prose_renderer |
| `m18_atuin_cache` | `m4_consolidation/m18_atuin_cache.rs` | L4 | Writes last successful injection payload to atuin KV as habitat.last-injection. Reads it back for Tier 2 fallback. Uses atuin kv set/get via subprocess (no Rust atuin SDK needed). | m01_types, m02_errors |
| `m19_preset_queries` | `m5_query/m19_preset_queries.rs` | L5 | Named query presets: trajectory (last 10), chains (unresolved by frequency), workstreams (active+blocked), patterns (top 20 by weight), health (latest per service). Each returns formatted text. | m01_types, m02_errors, m07_causal_chain, m08_trajectory, m09_workstream, m10_pattern |
| `m20_raw_query` | `m5_query/m20_raw_query.rs` | L5 | Raw SQL passthrough: takes arbitrary SQL string, executes against injection.db, returns formatted results (header + separator + rows). Safety: read-only connection (SQLITE_OPEN_READ_ONLY). | m01_types, m02_errors, m06_schema |
| `m21_fzf_browser` | `m5_query/m21_fzf_browser.rs` | L5 | Interactive fzf-powered memory browser. Pipes table contents through fzf with --preview showing related records. Requires fzf in PATH. Falls back to non-interactive preset display if fzf absent. | m01_types, m19_preset_queries |
| `m22_stdb_module` | `m6_stdb/m22_stdb_module.rs` | L6 | SpaceTimeDB WASM module: 5 table definitions (Rust #[spacetimedb::table]), 6 reducers (ingest_event, reinforce_edge, capture_gradient, register_session, run_decay, forget_sphere), schedule tables for decay + cache rebuild. Compiles to wasm32-unknown-unknown. | m01_types |
| `m23_ingester` | `m6_stdb/m23_ingester.rs` | L6 | Multi-source ingester binary: polls ORAC (30s), subscribes PV2 /bus/ws, polls SYNTHEX (60s), syncs POVM (300s). Calls STDB reducers via SDK. Exposes health on :3001. Long-running tokio process. | m01_types, m02_errors, m22_stdb_module |
| `m24_migration` | `m6_stdb/m24_migration.rs` | L6 | One-shot SQLite → STDB migration. Reads all 5 SQLite tables, calls STDB reducers to populate. Verification checksums (count + weight aggregates). Rollback on mismatch. | m01_types, m02_errors, m06_schema, m22_stdb_module |
